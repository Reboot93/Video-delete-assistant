# Video-delete-assistant
#片库清理助手

遍历指定文件夹的视频，并调用potplayer播放，提供简单的删除同名文件/文件夹/上级文件夹功能，可以跳过指定个数的视频

Traverse the video, call potplayer to play, provide a few simple delete options
